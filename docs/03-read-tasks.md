# Phase 2-1: 读取任务表格

当用户告知"填完了"后，AI 读取任务表格，获取待编译的库列表。

---

## 执行流程

AI 自行判断处理：

1. **找到表格** - 查找当天日期的 `libs/porting-tasks-YYYY-MM-DD.xlsx`
2. **读取数据** - 读取表格内容，跳过空行
3. **智能处理** - 自动识别和修复常见问题（如 URL 格式、版本格式等）
4. **输出确认** - 列出任务清单，用户确认

**原则**：除非缺少关键信息（库名、仓库地址），否则自行处理，不询问用户。

---

## 下一步

确认任务后，进入 **2-2 拉取代码** 阶段。
